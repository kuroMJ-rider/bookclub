## 📚 데이터 인류학 살롱 (Data Anthropology Salon)

> **"숫자 너머의 사람을 읽는 분석가들을 위한 독서 아카이브"**

부트캠퍼 MJ가 '어쩌다' 시작한 '데이터 인류학 살롱'은 단순한 독서 기록을 넘어, 텍스트 속에 숨겨진 인간의 패턴과 통찰을 수집하는 프라이빗 아카이브 프로젝트입니다.

---

### 🏛️ 북클럽 3대 운영 원칙
사용자의 인지적 부담을 줄이고 본질에 집중하기 위해 아래의 원칙을 준수합니다.

1. **관찰의 기록**: 단순한 감상이 아닌, 분석적 시각으로 문장 속의 데이터를 포착합니다.
2. **패턴의 발견**: 흩어진 문장들을 연결하여 하나의 인류학적 흐름을 찾아냅니다.
3. **지식의 공유**: 분석된 통찰을 노션(Notion) 데이터베이스로 실시간 동기화하여 지식 자산화합니다.

---

### 🛠️ 주요 기능 (Key Features)
* **실시간 노션 연동**: 앱에서 입력한 인상 깊은 문장이 Notion Database로 즉시 전송됩니다.
* **모바일 최적화**: 언제 어디서든 독서 중 떠오른 영감을 기록할 수 있도록 반응형 UI로 설계되었습니다.
* **데이터 무결성**: Supabase를 활용하여 수집된 데이터를 안정적으로 관리합니다.

---

### 💻 기술 스택 (Tech Stack)
* **Framework**: Next.js 15 (App Router)
* **Database**: Supabase
* **Integration**: Notion API
* **Styling**: Tailwind CSS
* **Development**: AI-Powered Development (Cursor)
* **Deployment**: Vercel
